# 実装計画: LP 大規模コンテンツ拡充＆クオリティ強化 (Phase 5)

本計画は、ユーザーからの厳格な指示と企画書データ（`第2回次世代自治共創会議_260420（案内版）.docx`）の完全解析に基づき、LPの「圧倒的なクオリティ」と「情報の厚み」を両立させるための大規模アップデート手順です。

## User Review Required

> [!IMPORTANT]
> 本実装により、ページ構成やテキストコンテンツが大幅にリッチ化されます。
> 以下の計画内容（特にManifestoのリライト方針や登壇者の階層構造）をご確認いただき、承認をお願いいたします。

## 修正・強化内容一覧

### 1. Hero Section の視覚的インパクト強化
- **[MODIFY]** `src/components/features/Hero.tsx`
  - タイトルの改行崩れを防ぐため、`whitespace-nowrap` とTailwindのレスポンシブなサイズ可変クラスを組み合わせた最適化を実施。
  - 背景レイヤーに「共創」を象徴する、抽象的でゆっくりと浮遊・交差する幾何学オーブ（`#7099c7`, `#f7b454`）をFramer Motionで実装し、スケール感を演出。

### 2. Manifesto のリライトと再構築 (パララックスと余白)
- **[MODIFY]** `src/components/features/Manifesto.tsx`
  - **コピーライティング**: 「補助金に依存しすぎない地方創生」「自治の未来を創る共助の力」などのエッジの効いた見出しと、プロ目線で洗練されたリード文へ完全リライト。
  - **デザイン**: `py-40` 以上の贅沢な余白と、巨大なタイポグラフィを採用。スクロールに応じてテキストが奥から手前へ迫る（`scale` と `y` のパララックス）3D的アニメーションを実装。

### 3. Speaker Grid の完全実装とプロフィールの網羅
- **[MODIFY]** `src/components/features/Speakers.tsx`
  - **登壇者の網羅と階層化**: 企画書から全10名を抽出。
    - *Tier 1 (基調講演)*: 鈴木寛氏、湯崎英彦氏、森田浩司氏（大サイズカード）
    - *Tier 2 (各部登壇・司会)*: 荒川裕貴氏、村上敬亮氏、小玉祥平氏、田島楓氏、横山裕一氏（中サイズカード）
    - *Tier 3 (所長等)*: 中山昌生氏、松原直輝氏（リストまたは小カード）
  - **プロフィール機能**: 各カードホバー時に、役職や専門性（「医療人材」「放課後共創」等）を解説するプロフィール文がグラスモーフィズムのレイヤーとして美しく浮き上がるインタラクションを追加。

### 4. Session Content の詳細化
- **[MODIFY]** `src/components/features/TimeTable.tsx`
  - 単なる文字の羅列を廃止し、Lucide Reactのアイコン（医療、教育、交通等）を交えたリッチなリスト形式へ拡張。
  - 「津南町の医療人材」「須崎市の教育人材」「まちの移動をつくりだす」など、具体的なトピックを視覚的にハイライトし、参加メリットを明示。

### 5. コンバージョンフローと情報の完結
- **[NEW / MODIFY]** `src/app/page.tsx`, `src/components/features/CTA.tsx`
  - TimeTableの下に、LPの目的である「プレエントリーはこちら」へ誘導する巨大なCTAコンポーネントを新規作成。
  - その直下に、主催（東京大学鈴木寛研究室等）と共催（懐徳会等）の情報を、洗練されたタイポグラフィでレイアウト。
- **[MODIFY]** `src/components/features/DataStats.tsx`
  - 昨年度実績セクション内に、セカンダリデザインの「昨年度の様子はこちら」ボタン（特設サイト `https://www.kaitoku.org/suzuden0531/` へのリンク）を追加。

# Walkthrough: UI/UX オーディット結果に基づくリファクタリング

批判的監査レポート（`improvement_point.md`）の指摘事項に沿って、LP全体のクオリティを底上げするリファクタリングを完遂しました。

## 実装のハイライト

> [!TIP]
> ブラウザ（`http://localhost:3000`）をリロードして、実際のモーションと質感をご確認ください。

### 1. アーキテクチャのクリーンアップ
- `src/lib/animations.ts` を新設し、`TRANSITIONS` や `EASING.apple` などのアニメーション定数を一元管理しました。これにより、各コンポーネント内に散在していたマジックナンバーが排除され、運用保守性と一貫性が向上しています。

### 2. 没入感（Vibe）の向上
- **FluidBackground**: `linear` イージングと長い `duration` を用いて、色がランダムな多角形軌道を描く有機的な流体アニメーションに変更しました。ノイズの opacity と `mix-blend-overlay` を調整し、高級感のあるテクスチャを実現しています。
- **Glassmorphism**: `Header.tsx` および `Speakers.tsx` のカードに、エッジハイライト（`inset` シャドウ）と美しいドロップシャドウを追加し、平面的なデザインから現実のガラスのような立体感を持たせました。

### 3. モーションの洗練
- **TimeTable**: 安っぽい `animate-pulse` を廃止し、Framer Motion の `scale` と `opacity` を用いた「脈打つような有機的な Glow エフェクト」を実装しました。
- **Hero**: スクロールインジケーター（縦線）に対して、`transformOrigin` を動的に変更することで「重力に引かれて伸び、下へ落ちていく」心地よいモーションを付与しました。

### 4. タイポグラフィと空間デザイン
- **Hero**: メインタイトル「次世代自治共創会議」に広めのトラッキング（`[0.15em]`）を持たせ、堂々とした佇まいに変更しました。サブタイトル「第2回」はサイズを縮小し、Trust Blueを適用することでジャンプ率を最大化しました。
- **Manifesto**: 背景に巨大な「MANIFESTO」の透かし文字を配置し、`useScroll` と `useTransform` を用いたパララックス（視差効果）を実装。セクション全体に圧倒的な奥行きを持たせました。また、見出し下には1pxの美しい装飾ラインを追加しています。

## 動作確認結果
- `npm run build` にて、TypeScriptエラーやビルドエラーが一切ないことを確認済みです。
- すべてのアニメーションが意図したAppleライクなイージング（`cubic-bezier(0.16, 1, 0.3, 1)`）で動作していることを確認しました。
